mlcf.datatools.standardisation module#

Standardisation Function Module.

This module provide the StandardisationModule class which provide methods compatible with DataIntevals and WTSeries to perform standardisation on features.

Bases: mlcf.datatools.standardisation.StandardisationModule

Allows to perform a standardisation with sklearn.preprocessing.MinMaxScaler. Transform features by scaling each feature to a given range.

__init__(minmax: Tuple[float, float] = (0, 1), feature_range: Tuple[float, float] = (0, 1))#

The features will be scale in the range given by {feature_range}. Instead of using the fit and partial fit methods, we can set the min and max values with the {minmax} parameter.

Parameters

  • minmax (Tuple[float, float], optional) – It allows to avoid the fit and partial_fit methods by setting the minimum and maximum value. Defaults to (0, 1).

  • feature_range (Tuple[float, float], optional) – The future range over which features will be scaled. Defaults to (0, 1).

class mlcf.datatools.standardisation.StandardisationModule(class_std: Callable, *args, **kwargs)#

Bases: abc.ABC

The Standardisation Module abstract class.

This abstract class is used as a basis for implementing interfaces to sklearn’s standardisation classes. The interest of having these interfaces allows to be compatible with pandas.DataFrame and to reshape the data.
